Telegram бот для анализа учебных данных из Excel-таблиц. 
Вохможности бота:
Бот анализирует данные по 6 направлениям - Расписание группы, темы занятий, успеваемость студентов, посещаемость, проверка ДЗ(у преподавателей), сдача ДЗ(у студентов).
Иструкция по использованию бота:
1) Запускаем бота в чате командой /start
2) В главном меню из 6 кнопок выбираем нужное направление.
3) Загружаем файл с данными под это направление.
4) Получаем результат.
5) Чтобы вернуться к выбору направления, так же присутствуют кнопки выхода в меня и стрелочки "назад".
